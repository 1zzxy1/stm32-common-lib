# LCD_Menu 按键菜单系统

> ✅ **通用模块** - 适用于STM32平台

## 功能特性

- 多级菜单结构
- 按键导航（上下左右确认）
- 参数编辑
- OLED/LCD显示适配
- 轻量级设计

## 使用场景

✅ **适用于**：参数调节、功能选择、调试界面
⚠️ **不适用于**：复杂UI（推荐WouoUI）

## API示例

```c
#include "KeyBoard.h"

// 初始化菜单
Menu_Init();

// 主循环处理
while(1)
{
    Menu_Process();  // 处理按键和显示
    delay_ms(10);
}
```

## 依赖项

- 按键驱动
- OLED/LCD显示驱动
