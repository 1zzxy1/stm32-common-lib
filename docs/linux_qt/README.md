## 学习路径建议

1. **Linux基础** → 阅读"linux简介"和"嵌入式基础"
2. **环境搭建** → 参考Ubuntu安装文档
3. **Qt入门** → 学习信号槽、主窗口、控件使用
4. **进阶功能** → 多线程、网络、数据库、文件IO
5. **实战练习** → 参考实验指导书完成实验

## 注意事项

1. **Qt版本**：文档基于Qt4/Qt5，注意API差异
2. **系统环境**：建议使用Ubuntu 16.04或更高版本
3. **依赖安装**：需要安装Qt开发库和工具链
4. **编码问题**：注意源码文件编码为UTF-8
5. **交叉编译**：嵌入式开发需配置交叉编译环境
