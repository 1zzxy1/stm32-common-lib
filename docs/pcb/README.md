## PCB设计基础知识

### 设计流程

1. **需求分析** → 确定电路功能和性能指标
2. **原理图设计** → 绘制电路原理图
3. **元件选型** → 选择合适的元器件
4. **PCB布局** → 规划元件位置
5. **PCB布线** → 连接电路
6. **设计审查** → 检查DRC/ERC规则
7. **生成文件** → 导出Gerber文件
8. **打样生产** → 发给PCB厂商制作

### 设计要点

1. **层数选择**：
   - 2层板：简单电路
   - 4层板：一般数字电路（带电源层和地层）
   - 6层及以上：高速电路、射频电路

2. **布线规则**：
   - 电源线加粗（>20mil）
   - 高速信号短且直
   - 差分线等长等距
   - 避免直角走线

3. **安全间距**：
   - 焊盘到焊盘：≥8mil
   - 走线到走线：≥8mil
   - 走线到焊盘：≥8mil

4. **过孔设置**：
   - 信号过孔：0.3mm孔径
   - 电源过孔：0.5mm或更大

### 常见问题

**原理图阶段**：
- [ ] 电源和地是否连接正确
- [ ] 信号是否有上拉/下拉电阻
- [ ] 去耦电容是否足够
- [ ] 晶振电路是否有负载电容

**PCB阶段**：
- [ ] 元件封装是否正确
- [ ] 高频信号是否远离电源
- [ ] 接口和连接器位置是否合理
- [ ] DRC检查是否全部通过

## 注意事项

1. **版权问题**：
   - 商业项目请注意原理图版权
   - 学习参考请标注来源

2. **安全规范**：
   - 注意高压电路安全间距
   - 遵守相关安全标准（如UL、CE）

3. **制造工艺**：
   - 了解PCB厂商的工艺能力
   - 设计时考虑可制造性（DFM）

4. **成本控制**：
   - 合理选择PCB层数
   - 避免使用特殊工艺（盲埋孔等）
