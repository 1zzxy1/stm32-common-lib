## 使用场景

✅ **适用于**：
- JavaWeb开发学习
- Web应用快速开发参考
- Session/Cookie机制理解
- Excel数据导入导出
- 实时通信功能实现
- 验证码/二维码集成

## 技术栈

- **后端**：Java Servlet / JSP
- **前端**：JavaScript / HTML
- **库依赖**：
  - Apache POI（Excel操作）
  - Zxing（二维码）
  - WebSocket API

## 使用方法

1. 解压对应的`.rar`或`.zip`文件
2. 导入到Eclipse/IDEA等IDE
3. 配置Tomcat服务器
4. 参考各示例中的代码和注释
5. 根据需求修改和集成到自己的项目

## 注意事项

1. **JDK版本**：建议使用JDK 8或以上
2. **Tomcat版本**：根据JDK版本选择对应的Tomcat
3. **依赖管理**：部分示例需要额外引入jar包
4. **编码问题**：注意设置项目编码为UTF-8避免中文乱码
5. **端口冲突**：确保Tomcat默认端口8080未被占用
