- **技术框架**：JSP、Servlet
- **前端技术**：JavaScript、jQuery、Ajax
- **数据库**：SQL多表关联

## 使用建议

1. **按需查阅**：根据遇到的问题查找对应文档
2. **环境配置**：先阅读"搭建开发环境"文档
3. **版本匹配**：注意JDK、JSP、Tomcat版本对应关系
4. **编码问题**：重点关注页面传值与编码文档
5. **部署参考**：学习WAR包导出和部署流程

## 注意事项

1. 文档基于JavaWeb传统开发方式（非Spring Boot）
2. 部分技术可能较老，但基础概念仍然适用
3. 使用前请确认技术栈版本兼容性
4. 建议结合实际项目理解文档内容
